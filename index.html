เชื่อมต่อ Google Sheets

กรอก URL ของ Google Apps Script Web App

วิธีตั้งค่า Google Apps Script:

  1. เปิด Google Sheets แล้วไปที่ Extensions → Apps Script
  2. วางโค้ด doPost(e) สำหรับรับข้อมูล
  3. Deploy เป็น Web App (Execute as: Me, Who has access: Anyone)
  4. คัดลอก URL แล้วนำมาวางด้านบน
หมายเหตุ: ตัวอย่าง Apps Script โค้ดที่ต้องใช้อยู่ในความคิดเห็นของไฟล์นี้
